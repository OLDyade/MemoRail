# Glossary

| Term | Meaning |
| --- | --- |
| fact source | Durable Markdown memory files. |
| derived index | Rebuildable runtime files such as `MEMORY.md`, `memory.json`, `entities.json`, and `bm25.json`. |
| memory root | Directory that contains MemoRail memory files and derived indexes. |
| context prelude | Runtime prompt block built from relevant memories. |
| native adapter | Host-specific lifecycle integration for agents such as OpenCode, OpenClaw, Hermes, Pi Agent, or Codex. |
| doctor | Read-only health check for schema, path, lock, and derived-index problems. |
| dream | Future offline consolidation process for duplicates, stale facts, and conflicts. |
